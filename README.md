# Diwali-sales-analysis
📊 Diwali Sales Analysis (Exploratory Data Analysis)
📌 Project Overview

This project focuses on analyzing Diwali sales data using Exploratory Data Analysis (EDA). The goal is to understand customer purchasing behavior, identify sales trends, and extract meaningful business insights that can help improve marketing strategies and decision-making.

🎯 Objectives
Perform data cleaning and preprocessing
Analyze customer demographics and behavior
Identify sales trends across categories and regions
Visualize data to uncover hidden patterns
Generate actionable business insights
🛠️ Tools & Technologies
Programming Language: Python
Libraries: Pandas, NumPy, Matplotlib, Seaborn
Environment: Jupyter Notebook / Google Colab
Version Control: Git & GitHub
Data Format: CSV / Excel
📂 Dataset Description

The dataset contains information about customer purchases during the Diwali season, including:

Customer demographics (age, gender, marital status, occupation)
Product categories
Purchase amounts
Region/zone details
⚙️ Project Workflow
Data Collection
Imported dataset into Python environment
Data Cleaning
Handled missing values
Removed duplicates
Corrected data types
Data Preprocessing
Feature selection and transformation
Data filtering and grouping
Exploratory Data Analysis (EDA)
Analyzed customer demographics
Studied sales patterns across categories
Examined regional performance
Data Visualization
Used Seaborn and Matplotlib to create charts such as:
Bar plots
Count plots
Histograms
Heatmaps
Insights Generation
Identified key trends and high-performing segments
Derived business recommendations
📈 Key Insights
Certain customer groups contribute more to overall sales
Specific product categories perform better during festive seasons
Regional differences significantly impact purchasing behavior
Targeted marketing can improve conversion rates
